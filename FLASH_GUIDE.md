# RK3128 TP-9038 刷机指南

## 所需工具
1. Rockchip 刷机工具 (RKDevTool)
2. USB 转 TTL 串口线（可选，用于调试）
3. 镊子或跳线（用于短接进入刷机模式）

## 进入刷机模式
1. 拆开电视盒子
2. 找到 Flash 芯片附近的测试点（通常标有 CLK、CMD、D0、GND）
3. 短接 CLK 和 GND（或按照板子上的提示）
4. 连接 USB 到电脑
5. 保持短接状态上电

## 刷机步骤
1. 打开 RKDevTool
2. 选择下载好的固件（通常是 `.img` 或 `.img.gz` 文件）
3. 点击"执行"
4. 等待刷机完成（进度条走完）
5. 断开电源，移除短接

## 首次启动
1. 连接网线到路由器
2. 上电启动
3. 等待 2-3 分钟
4. 在路由器管理页面查看设备的 IP 地址
5. 浏览器访问：http://[设备IP]

## 默认设置
- 用户名：root
- 密码：password
- IP 地址：DHCP 自动获取
- 无线网络：OpenWrt（无密码）

## 打印机配置
1. 登录 LuCI 管理界面
2. 转到"服务" -> "CUPS"
3. 添加打印机（自动检测或手动）
4. 配置共享设置
